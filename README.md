A Basic Rails Todo List
-----------------------

Purpose
=======

This is a learning project for NSS Cohort 2.

Usage:
======

This is a standard rails app. To run tests, use 'rake'

Features:
=========

1. Enter a list of tasks
2. Edit tasks
3. Multiple lists
4. Reorder tasks
5. Finish tasks
